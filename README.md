# 🎬 Advanced YouTube Downloader

> *A feature-rich desktop application to download YouTube videos and playlists with high quality, metadata embedding, and subtitle support.*

[![Python](https://img.shields.io/badge/Python-3.7+-blue?logo=python)](https://python.org)
[![PyQt5](https://img.shields.io/badge/GUI-PyQt5-41CD52?logo=qt)](https://riverbankcomputing.com/software/pyqt/)
[![yt-dlp](https://img.shields.io/badge/Backend-yt--dlp-FF0000?logo=youtube)](https://github.com/yt-dlp/yt-dlp)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

**Advanced YouTube Downloader** is a desktop application built with **PyQt5** and **yt-dlp**. It allows you to download videos, playlists, audio, subtitles, and thumbnails — all with a clean, responsive interface. Choose from multiple resolutions (360p up to 8K), output formats, and even embed metadata and thumbnails into your files.

---

## ✨ Features

| Category | Details |
|----------|---------|
| 📥 **Downloads** | Single videos or entire playlists |
| 🎞️ **Quality** | 360p, 480p, 720p, 1080p, 1440p, 2160p (4K), 4320p (8K) + “Best Available” |
| 🎧 **Content Type** | Video+Audio, Video Only, Audio Only |
| 🎬 **Formats** | MP4, MKV, WEBM (video) – MP3, AAC, M4A, OPUS, FLAC (audio) |
| 📝 **Subtitles** | Download and embed subtitles (all available languages) |
| 🖼️ **Thumbnails** | Download thumbnail separately or embed into video |
| 🏷️ **Metadata** | Add custom metadata (title, artist, album, etc.) to files |
| 🍪 **Cookies** | Use Netscape cookies file or browser cookies to access restricted content |
| 📁 **Playlist Range** | Download a specific range (e.g., `1-5` or `1,3,5`) |
| 🌗 **Themes** | Toggle between light and dark mode |
| 📊 **Progress** | Real‑time progress bar + detailed log console |
| ❌ **Cancel** | Close the app with confirmation to stop ongoing downloads |
